## Product Listing Demo Project
### Description
It's a demo website for the product listing which is made by Next.js, React, and Styled-Components
### Keywords
Next.js, React, Styled-Components, e-commerce
### Usages

* Installation

```

npm install

```

* Launch

```

npm run dev

```

### Stacks
* Next.js
* ReactJS
* Redux (Redux-thunk)
* Styled-components
